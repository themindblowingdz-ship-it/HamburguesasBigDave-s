## Alumno
**David Zamora**

## Descripción del Proyecto
Esta aplicación web simula un sistema de pedidos para un restaurante de hamburguesas llamado **Big Dave’s Burger**.  
El proyecto integra conceptos de **HTML semántico, CSS responsivo, JavaScript interactivo** y una **simulación de nube mediante `localStorage`**, cumpliendo con los requisitos técnicos.

Los usuarios pueden:  
- Explorar el menú con hamburguesas, papas fritas y refrescos en tamaños **Chico, Mediano y Grande**.  
- Agregar productos al carrito y ver el total acumulado.  
- Finalizar el pedido y ver una pantalla de confirmación con el total pagado y una imagen del repartidor.  
- Navegar entre secciones: **Inicio, Menú y Contacto**.

## Funcionalidades Destacadas

- **HTML semántico:** Uso de `<header>`, `<main>`, `<section>`, `<footer>` para una estructura clara.  
- **CSS responsivo:** Diseño adaptable a pantallas grandes y móviles mediante `@media queries`.  
- **JavaScript interactivo:**  
  - Agrega productos al carrito con botones.  
  - Calcula el total acumulado.  
  - Muestra la confirmación del pedido.  
- **Simulación de nube:**  
  - Uso de `localStorage` para guardar los datos del carrito y total, simulando almacenamiento en servidor, la principal señal de simulación en la nube es que al  cambiar de pestaña el pedido sigue intacto, sencillo pero efectivo. En un caso real que no fuera simulación en la nube la info quedaría almacenada para saber directamente qué producto se consume más.
- **Diseño y usabilidad:**  
  - Navegación clara por secciones.  
  - Botones y listas centradas y accesibles.  
  - Colores consistentes con la temática (rojo, amarillo y blanco).

## Instrucciones de Uso

1. Abrir el proyecto en un navegador moderno mediante [GitHub Pages](https://themindblowingdz-ship-it.github.io/HamburguesasBigDave-s/).  
2. Navegar entre las secciones usando los botones del **header**:  
   - **Inicio:** Bienvenida y banner del restaurante.  
   - **Menú:** Arma tu combo, selecciona tamaños y agrega al carrito.  
   - **Contacto:** Información de contacto.  
3. Agregar productos al carrito y presionar **Finalizar Pago** para ver la confirmación del pedido y el total pagado.  
4. Presionar **Volver al Inicio** para reiniciar el pedido.
